# @rosarium/liturgy-engine

Motor de cálculo litúrgico e calendário da Igreja Católica para o projeto Rosarium.

Esta biblioteca encapsula toda a complexidade de definição do Ano Litúrgico, precedência de festas, cálculo da Páscoa e determinação das leituras diárias.

## Instalação

Como faz parte do monorepo Rosarium, ela será referenciada e instalada via workspace.
